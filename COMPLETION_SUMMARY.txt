🚀 HOW TO RUN

Quick Start (5 minutes):
  1. pip install -r requirements.txt
  2. cp .env.example .env
  3. Add GEMINI_API_KEY to .env
  4. streamlit run app.py
  5. Open http://localhost:8501

Verification:
  python verify.py  # Check dependencies

Configuration:
  Edit .env file with:
  - GEMINI_API_KEY (required)
  - AFFERENS_API_KEY (optional)
  - CAMERA_INDEX, FPS, etc (optional)

🎯 FOCUS SCORING ALGORITHM

Base Score: 100

Adjustments:
  • Distraction detected:        -15 per object
  • Not looking at screen:       -20
  • High fatigue:                -25
  • Study materials present:      +5
  • Input devices visible:        +3

Final: MAX(0, MIN(100, adjusted))

Classification:
  • High (80-100):  🟢 Excellent focus
  • Medium (50-79): 🟡 Good progress
  • Low (0-49):     🔴 Needs improvement
